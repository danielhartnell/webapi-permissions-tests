Hosted App Updates - Appcache Preloading

Simple Appcache Update

Tests a simple update of the appcache from a very small amount of appcache resources to slightly more. Directions for testing this is below.

  1. Install the hosted app with the button below
  2. Copy the appcache data from state_medium_appcache.appcache into test.appcache
  3. Push the changes to test.appcache to the server
  4. When updates fire, confirm and update the app
  5. Uninstall the app
  6. Copy the appcache data from state_small_appcache.appcache into test.appcache
  7. Push the changes to test.appcache to the server

Large Appcache Update

Tests a simple update of the appcache from a very small amount of appcache resources to a lot more. Directions for testing this is below.

  1. Install the hosted app with the button below
  2. Change the appcache manifest to point to the large appcache instead of the small appcache
  3. Check for updates manually
  4. When updates fire, confirm and update the app
  5. Uninstall the app
  6. Switch the app back to using the old appcache
